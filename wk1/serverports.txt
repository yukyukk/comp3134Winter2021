•What is the definition of server port?
A server port is a number which represents an interface from a web appication to a server.

•What is server port range?
Server port ranges are classifications of server ports depending on the kind of use they will see.
The different server port ranges are Well-Known(0-1023), Registered(1024-49151), and Dynamic(49152-65535).

Unless you meant what is the range of ports, then it would be 0-65535.
